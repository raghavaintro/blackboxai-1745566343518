How to Add Your Game Files

To add your game files for download, follow these steps:

  1. Upload your game files to the downloads/ folder in this project workspace.
  2. Rename your files appropriately (e.g., mygame.zip).
  3. Add a new game card in the index.html file under the "Available Games" section with the correct file link.
  4. Example of a game card: 
    
  
    My Game Title
    
  
    Description of my game.
    
  
     Download

Once you add your files and update the HTML, visitors to your website will be able to download your games for free.
